Capteur PIR (Infrarouge Passif) :

Les capteurs PIR détectent les changements dans le rayonnement infrarouge émis par les objets en mouvement dans leur champ de vision.
Lorsqu'un objet chaud (comme le corps humain) traverse le champ de vision du capteur, il détecte le changement de chaleur et active l'alarme.
Les capteurs PIR sont efficaces pour détecter les mouvements à courte distance et sont souvent utilisés dans les systèmes d'alarme résidentiels.
Capteurs volumétriques :

Les capteurs volumétriques utilisent différentes technologies, telles que les ultrasons, les micro-ondes ou la lumière, pour détecter les mouvements dans une zone spécifique.
Contrairement aux capteurs PIR, les capteurs volumétriques peuvent détecter les mouvements dans une zone plus large et peuvent parfois fonctionner à travers des obstacles tels que les murs.
Les capteurs volumétriques peuvent être utilisés pour surveiller de grandes zones telles que les salles de séjour, les couloirs, etc.
Outre ces deux options, il existe d'autres capteurs de présence disponibles sur le marché, tels que les capteurs à ultrasons et les capteurs à micro-ondes. Chaque type de capteur présente des avantages et des inconvénients en fonction des besoins spécifiques de votre système de sécurité résidentiel. Il est important de choisir celui qui convient le mieux à votre application et à votre budget. Vous pouvez également envisager de combiner plusieurs types de capteurs pour une détection plus robuste et fiable.
